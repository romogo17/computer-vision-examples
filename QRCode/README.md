# QR Code

> Taken from [LearnOpenCV](https://www.learnopencv.com/)

## Run the examples

For **Python**:

```sh
python qr_code.py <image_path>
```

For **C++**, first build the examples and then:

```sh
./qr_code <image_path>
```

## Build the C++ examples

Specify the OpenCV_DIR in CMakeLists.txt file. Then:

```sh
mkdir build
cd build
cmake ..
make
```
